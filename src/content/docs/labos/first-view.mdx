## Création des vues

Avant de créer les vues, consultez la documentation :  
[Odoo Views Documentation](https://www.odoo.com/documentation/17.0/developer/reference/addons/views.html)

Créez dans le dossier `views` :

### 1. Fichier `todo_menu.xml`

Ajoutez un menu permettant d’accéder à la liste et au formulaire des tâches.

```xml
<odoo>
  <menuitem id="menu_todo_root" name="To-Do"/>
  <record id="action_todo_task" model="ir.actions.act_window">
    <field name="name">Tasks</field>
    <field name="res_model">todo.task</field>
    <field name="view_mode">tree,form</field>
  </record>
  <menuitem id="menu_todo_task" name="Tasks" parent="menu_todo_root" action="action_todo_task"/>
</odoo>
```

Ajoutez ce fichier dans la section `data` du manifest.

### 2. Fichier `todo_view.xml`

```xml
        <group>
          <field name="name"/>
          <field name="is_done"/>
          <field name="date_deadline"/>
          <field name="team_ids"/>
        </group>
      </sheet>
    </form>
  </field>
</record>
```

#### Liste

```xml

      <field name="user_id"/>
      <field name="date_deadline"/>

  </field>
</record>
```

```xml
  </field>
</record>
```
